
# 결과 저장 데이터 정의서: 가동률_결과저장.xlsx
이 파일은 분석된 가동률 및 진료효율 데이터를 날짜별로 저장하는 결과 리포트 시트입니다.

## 1. [yyyymmdd가동률양식] 시트
새로운 날짜의 데이터를 생성할 때 복사해서 사용하는 원본 템플릿입니다.

### 주요 데이터 영역 및 셀 위치:
1. **상단 요약 (A1:E4)**
   - 합계, 1층, 2층별로 [가동률, 목표시간, 진료효율, 진료시간]을 요약 표시합니다.
2. **시간대별 가동률 및 대기 (A7:K20)**
   - 시작시간 08:00 ~ 20:00까지 1시간 단위 분석.
   - **가동시간**: 해당 시간대에 겹치는 '목표시간(AU열)'의 합계.
   - **가동률**: (가동시간 / 해당 시간대 가용 시간) * 100.
3. **직원별 지표 (A22:E32)**
   - 각 직원별 진료시간, 가동률, 진료효율, 복합청구 건수를 기록합니다.
4. **원장별 지표 (H22:M28)**
   - 각 원장별 진료시간, 가동률, 진료효율, 복합청구 비율을 기록합니다.

## 2. 날짜별 결과 시트 (예: 2025-12-23(화)가동률)
- 위의 양식을 복사하여 데이터가 채워진 실측 리포트 시트입니다.

## 3. 날짜별 상세 접수기록 시트 (예: 2025-12-23(화)접수기록)
- 원본 엑셀(A:AN열) 데이터에 분석 엔진이 계산한 AO~BC열 데이터가 추가된 중간 과정 시트입니다.
- **AU열 (목표시간)**: 계산 엔진이 산출한 표준 진료 시간.
- **AW열 (진료효율)**: 목표시간 / 실제진료시간 * 100.
- **BC열 (보험/비보험)**: '보험만', '비보험만', '보험+비보험' 분류 결과.





# 결과 저장 데이터 정의서: 가동률_결과저장.xlsx (상세 매핑 포함)

## 1. [yyyymmdd가동률양식] 상세 셀 매핑
분석된 데이터는 새 시트의 다음 좌표에 정확히 입력되어야 합니다.

### A. 상단 요약 테이블 (A1:E4)
| 항목 | 목표시간(B) | 진료효율(C) | 실제진료시간(D) | 가동률(E) |
| :--- | :--- | :--- | :--- | :--- |
| **합(Row 2)** | 전체 기준시간 합 | 전체 평균 효율 | 전체 실제시간 합 | 전체 가동률 |
| **1층(Row 3)** | 1층 기준시간 합 | 1층 평균 효율 | 1층 실제시간 합 | 1층 가동률 |
| **2층(Row 4)** | 2층 기준시간 합 | 2층 평균 효율 | 2층 실제시간 합 | 2층 가동률 |

### B. 시간대별 가동률 (A9:E20)
* 시작 시간(J열) 기준으로 해당 시간대(8:00~20:00)를 찾아 입력합니다.
- **C열 (가동시간)**: 해당 시간대에 진행된 진료들의 '목표시간(AU)' 합계
- **D열 (가동률)**: (C열 가동시간 / 요일별 1층+2층 시간당 가용시간) * 100
- **E열 (원인)**: 효율이 낮거나 높은 특이사항 기록 (비워둠)

### C. 직원별 가동률 및 효율 (A24:E32)
* A열에 적힌 이름(직원1, 직원2 등)을 찾아 해당 행에 입력합니다.
- **B열 (진료시간)**: 해당 직원의 실제 진료시간 합계 (AR열 합계)
- **C열 (가동률)**: (해당 직원 기준시간 합 / 요일별 개인 총 근무시간) * 100
- **D열 (목표가중치)**: 해당 직원의 기준시간 합계 (AU열 합계)
- **E열 (진료효율)**: (D열 / B열) * 100

### D. 원장별 가동률 및 효율 (H24:M28)
* H열에 적힌 이름(원장1, 원장2 등)을 찾아 해당 행에 입력합니다.
- **I열 (진료시간)**: 해당 원장의 실제 진료시간 합계 (AR열 합계)
- **J열 (가동률)**: (해당 원장 기준시간 합 / 요일별 개인 총 근무시간) * 100
- **K열 (목표가중치)**: 해당 원장의 기준시간 합계 (AU열 합계)
- **L열 (진료효율)**: (K열 / I열) * 100

---

## 2. 중간 과정 시트: [YYYY-MM-DD(요일)접수기록] 매핑
원본 데이터(A~AN열) 옆에 추가되는 계산 열 정의입니다.

- **AO열 (층구분)**: '층별체어' 시트 참조 (1층/2층)
- **AP열 (진료시작)**: 원본 J열 값
- **AQ열 (진료완료)**: 원본 K열 값
- **AR열 (진료시간)**: (AQ - AP) 분 단위 계산
- **AU열 (목표시간)**: 룰북 v1.7에 따른 기준시간 계산값
- **AW열 (진료효율)**: (AU / AR) * 100
- **BA열 (진료내용)**: 원본 P열 값
- **BC열 (보험/비보험)**: '보험만', '비보험만', '보험+비보험' 판별값
